# 🛍️ Myntra Review Scraper & Analysis

## 📌 Project Overview
This project focuses on scraping product reviews from Myntra and performing data analysis to extract meaningful insights about customer sentiment and product performance.

It is an end-to-end data project covering:
- Web scraping
- Data cleaning
- Exploratory data analysis (EDA)
- Insight generation

---

## 🚀 Features
- 🔍 Scrapes product reviews from Myntra
- 🧹 Cleans and preprocesses raw data
- 📊 Performs exploratory data analysis (EDA)
- 💡 Generates insights from customer feedback
- 📁 Organized project structure for scalability

---

## 🛠️ Tech Stack
- Python
- Web scraping libraries (to be defined)
- Pandas
- Data visualization tools (optional)
